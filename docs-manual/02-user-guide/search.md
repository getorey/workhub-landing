# 검색

Workhub 전체에서 메시지, 사용자, 채널, 프로젝트를 통합 검색합니다.

## 통합 검색

사이드바 > **검색**을 클릭하거나 상단 검색창을 사용합니다.

### 검색 대상

| 대상 | 설명 |
|------|------|
| 메시지 | 채널, 토픽, DM의 메시지 내용 |
| 사용자 | 이름, 이메일로 사용자 검색 |
| 채널 | 채널 이름으로 검색 |
| 프로젝트 | 프로젝트 이름으로 검색 |

### 검색 결과

- 검색어가 포함된 결과가 카테고리별로 표시됩니다.
- 결과를 클릭하면 해당 항목의 상세 화면으로 이동합니다.
- 메시지 검색 결과에서는 검색어가 하이라이트됩니다.
