# 연락처

조직 내 구성원의 연락처를 관리합니다.

## 연락처 목록

사이드바 > **연락처**를 클릭하면 연락처 목록이 표시됩니다.

- 이름, 부서, 직급 등으로 검색할 수 있습니다.
- 목록에서 사용자를 클릭하면 상세 정보를 확인합니다.

## 연락처 상세

사용자를 선택하면 다음 정보를 확인할 수 있습니다:

- 이름, 직급, 부서
- 이메일, 전화번호
- 상태 메시지

## 빠른 액션

연락처 상세에서 바로 다음 작업을 수행할 수 있습니다:

- **DM 보내기**: 해당 사용자와 1:1 대화를 시작합니다.

## 차단

특정 사용자의 DM을 차단할 수 있습니다.

1. 연락처 상세에서 **차단** 버튼을 클릭합니다.
2. 차단된 사용자는 DM을 보낼 수 없습니다.
3. 차단 해제는 동일한 위치에서 가능합니다.
