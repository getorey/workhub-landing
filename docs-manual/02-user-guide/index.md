# 사용자 가이드

1. [프로젝트](./projects.md) — 프로젝트 생성, 참여, 토픽 관리

3. [태스크](./tasks.md) — 칸반 보드, 간트 차트, 태스크 관리
4. [다이렉트 메시지](./dm.md) — 1:1 DM, 그룹 DM
5. [채널](./channels.md) — 채널 가입, 메시지, 설정
6. [연락처](./contacts.md) — 연락처 관리, 그룹, 차단
7. [조직도](./org-tree.md) — 조직 구조 조회, 구성원 검색
8. [검색](./search.md) — 통합 검색
9. [파일](./files.md) — 파일 업로드, 다운로드
