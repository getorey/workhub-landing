# 메시지

채널과 토픽에서 팀원과 메시지를 주고받을 수 있습니다.

## 메시지 보내기

1. 채널 또는 토픽의 상세 화면으로 이동합니다.
2. 하단의 입력창에 메시지를 작성합니다.
3. Enter 키 또는 전송 버튼을 클릭합니다.

## 쓰레드 (답글)

특정 메시지에 대한 답글을 쓰레드로 관리합니다.

1. 메시지에 마우스를 올리면 액션 버튼이 표시됩니다.
2. **답글** 아이콘을 클릭합니다.
3. 우측에 쓰레드 패널이 열리고, 해당 메시지에 대한 답글을 작성합니다.

## 멘션 (@)

메시지에서 `@이름`을 입력하면 해당 사용자에게 알림이 전송됩니다.

## 리액션

메시지에 이모지로 반응할 수 있습니다.

1. 메시지에 마우스를 올려 이모지 아이콘을 클릭합니다.
2. 원하는 이모지를 선택합니다.
3. 같은 이모지를 다시 클릭하면 취소됩니다.

## 메시지 고정

중요한 메시지를 고정하여 쉽게 찾을 수 있습니다.

1. 메시지의 **핀** 아이콘을 클릭합니다.
2. 고정된 메시지는 상단에 표시됩니다.

## 메시지 수정 및 삭제

- **수정**: 메시지 액션 메뉴 > 수정을 클릭하여 내용을 변경합니다.
- **삭제**: 메시지 액션 메뉴 > 삭제를 클릭합니다. 본인이 작성한 메시지만 삭제 가능합니다.

## 메시지에서 태스크 생성

메시지 내용을 기반으로 태스크를 바로 생성할 수 있습니다.

1. 메시지 액션 메뉴에서 **"태스크 만들기"**를 클릭합니다.
2. 태스크 제목과 상세 정보를 입력합니다.
3. 생성된 태스크는 해당 토픽의 태스크 보드에 추가됩니다.
