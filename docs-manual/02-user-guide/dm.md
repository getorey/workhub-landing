# 다이렉트 메시지 (DM)

팀원과 1:1 또는 그룹으로 비공개 대화를 할 수 있습니다.

## DM 목록

사이드바 > **다이렉트 메시지**를 클릭하면 진행 중인 DM 목록이 표시됩니다.

- 읽지 않은 메시지가 있는 DM은 **굵은 글씨**로 표시됩니다.
- 최근 메시지 순으로 정렬됩니다.

## 1:1 DM

### 새 DM 시작

1. DM 목록 화면에서 **"+ 새 메시지"** 버튼을 클릭합니다.
2. 대화 상대를 검색하여 선택합니다.
3. 메시지를 입력하고 전송합니다.

### 기존 DM 이어서 대화

DM 목록에서 대화를 클릭하면 이전 대화 내용을 확인하고 이어서 메시지를 보낼 수 있습니다.

## 그룹 DM

### 그룹 DM 생성

1. **"+ 새 메시지"** 버튼을 클릭합니다.
2. 참여자를 2명 이상 선택합니다.
3. 메시지를 입력하고 전송하면 그룹 DM이 생성됩니다.

### 그룹 DM 특징

- 참여자 전원에게 메시지가 전달됩니다.
- 참여자 목록은 DM 상세 화면 상단에서 확인할 수 있습니다.

## DM 기능

DM에서도 일반 메시지와 동일한 기능을 사용할 수 있습니다:

- **쓰레드**: 특정 메시지에 답글 달기
- **리액션**: 이모지 반응
- **멘션**: `@이름`으로 특정 참여자 호출
- **메시지 수정/삭제**: 본인 메시지 편집 및 삭제
