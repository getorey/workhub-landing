# 파일

메시지에 파일을 첨부하고 관리합니다.

## 파일 업로드

1. 메시지 입력창의 **파일 첨부** 아이콘을 클릭합니다.
2. 업로드할 파일을 선택합니다.
3. 파일이 메시지와 함께 전송됩니다.

### 드래그 앤 드롭

메시지 입력 영역에 파일을 직접 드래그하여 업로드할 수도 있습니다.

## 채널 파일 저장소

채널의 모든 파일을 웹하드처럼 편리하게 관리할 수 있습니다.

### 파일 탭 열기

1. 채널에서 오른쪽 패널의 **파일** 탭을 클릭합니다.
2. 채널에 업로드된 모든 파일이 목록으로 표시됩니다.

### 파일 검색

파일명 검색으로 원하는 파일을 빠르게 찾을 수 있습니다.

1. 검색 입력창에 파일명을 입력합니다.
2. 실시간으로 검색 결과가 필터링됩니다.

### 파일 유형 필터

유형별로 파일을 필터링할 수 있습니다.

- **전체**: 모든 파일
- **이미지**: JPG, PNG, GIF, SVG 등
- **문서**: PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX, TXT 등
- **미디어**: MP3, MP4, WAV 등
- **압축**: ZIP, RAR, TAR 등

### 파일 정렬

컬럼 헤더를 클릭하여 정렬할 수 있습니다.

- **파일**: 파일명순 정렬
- **크기**: 파일 크기순 정렬
- **등록일**: 업로드 날짜순 정렬

오름차순/내림차순은 헤더 클릭 시 토글됩니다.

## 파일 다운로드

### 개별 다운로드

메시지에 첨부된 파일을 클릭하면 다운로드됩니다. 파일 목록에서 다운로드 아이콘을 클릭해도 됩니다.

### 일괄 다운로드

여러 파일을 한 번에 다운로드할 수 있습니다.

1. 다운로드할 파일의 체크박스를 선택합니다.
2. 하단에 선택된 파일 개수가 표시됩니다.
3. **선택 다운로드** 버튼을 클릭합니다.
4. 선택한 파일들이 ZIP 압축파일로 다운로드됩니다.

::: tip
최대 20개 파일까지 일괄 다운로드 가능합니다.
:::

## 원본 메시지로 이동

파일 목록에서 파일이 첨부된 원본 메시지로 바로 이동할 수 있습니다.

1. 파일 행의 메시지 아이콘을 클릭합니다.
2. 채널 메시지 목록으로 스크롤 이동합니다.
3. 해당 메시지가 노란색으로 하이라이트됩니다.

## 지원 파일 형식

- 이미지: JPG, PNG, GIF, SVG
- 문서: PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX
- 기타: ZIP, TXT, CSV

## 파일 미리보기

이미지 파일은 메시지 내에서 미리보기가 표시됩니다. 클릭하면 원본 크기로 확인할 수 있습니다.
