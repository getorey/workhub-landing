# 태스크

토픽 내에서 태스크를 생성하고 관리합니다. 칸반 보드와 간트 차트 두 가지 뷰를 제공합니다.

## 태스크 보기

토픽 상세 화면 > **태스크** 탭을 선택합니다. 상단의 토글로 뷰를 전환합니다:

- **보드**: 칸반 스타일 (할 일 → 진행중 → 검토 → 완료)
- **간트차트**: 타임라인 기반 일정 뷰

## 태스크 생성

### 보드에서 생성

1. 원하는 상태 컬럼의 **"+"** 버튼을 클릭합니다.
2. 제목, 설명, 우선순위, 담당자, 마감일을 입력합니다.

### 메시지에서 생성

메시지 액션 메뉴 > **"태스크 만들기"**를 클릭하면 메시지 내용이 태스크 설명에 자동 입력됩니다.

## 태스크 상태

| 상태 | 설명 |
|------|------|
| 할 일 (todo) | 아직 시작하지 않은 태스크 |
| 진행중 (in_progress) | 현재 작업 중 |
| 검토 (review) | 검토 대기 중 |
| 완료 (done) | 완료된 태스크 |

## 태스크 우선순위

| 우선순위 | 색상 |
|----------|------|
| 긴급 | 빨강 |
| 높음 | 주황 |
| 보통 | 파랑 |
| 낮음 | 회색 |

## 간트 차트

### 타임라인 뷰

- **일별 / 주별 / 월별** 줌 레벨을 전환할 수 있습니다.
- 오늘 날짜는 빨간 점선으로 표시됩니다.
- **오늘로 이동** 버튼으로 현재 날짜 위치로 스크롤합니다.

### 태스크 바 드래그

- **좌우 드래그**: 태스크의 시작일/종료일을 변경합니다.
- **좌우 끝 드래그**: 시작일 또는 종료일만 개별 변경합니다.

### 순서 변경

좌측 태스크 목록에서 드래그 핸들(⠿)을 잡고 위아래로 드래그하면 순서가 변경됩니다.
변경된 순서는 서버에 자동 저장됩니다.

### 태스크 편집

태스크 이름을 클릭하면 편집 모달이 열립니다:

- 제목, 설명, 상태, 우선순위, 날짜 수정
- 마일스톤 설정
- **의존성 관리**: 선행 태스크를 추가/삭제할 수 있습니다.

### 의존성 (선행 태스크)

태스크 간의 선후 관계를 설정하면 간트 차트에 화살표로 표시됩니다.

1. 태스크 편집 모달에서 **의존성** 섹션의 **"추가"** 버튼을 클릭합니다.
2. 선행 태스크를 선택합니다.
3. 삭제하려면 의존성 목록의 휴지통 아이콘을 클릭합니다.
