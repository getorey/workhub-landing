# 채널

채널은 조직 내 주제별 공개 대화 공간입니다. 프로젝트와 달리 태스크 관리 없이 메시지 중심으로 운영됩니다.

## 채널 목록

사이드바 > **채널**을 클릭하면 참여 중인 채널과 가입 가능한 채널 목록이 표시됩니다.

## 채널 가입

1. 채널 목록에서 가입하지 않은 채널을 확인합니다.
2. **"가입"** 버튼을 클릭합니다.
3. 가입 후 해당 채널의 메시지를 확인하고 대화에 참여할 수 있습니다.

## 채널 메시지

채널 내 메시지 기능은 토픽 메시지와 동일합니다:

- 메시지 전송 (텍스트, 파일 첨부)
- 쓰레드 (답글)
- 멘션 (`@이름`)
- 리액션 (이모지)
- 메시지 고정
- 메시지 수정 및 삭제

## 채널 설정

채널 관리자는 다음을 설정할 수 있습니다:

- **채널 이름 변경**
- **채널 설명 수정**
- **채널 보관**: 더 이상 사용하지 않는 채널을 보관 처리합니다.
