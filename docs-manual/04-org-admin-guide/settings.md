# 시스템 설정

기관 관리자는 조직의 시스템 연동 및 정책을 설정합니다.

## SMTP 설정

이메일 알림 발송을 위한 SMTP 서버를 설정합니다.

| 설정 | 설명 |
|------|------|
| SMTP 서버 | 메일 서버 주소 |
| 포트 | SMTP 포트 (587 권장) |
| 암호화 | TLS / STARTTLS |
| 사용자 | SMTP 인증 계정 |
| 비밀번호 | SMTP 인증 비밀번호 |
| 발신자 이메일 | 알림 메일의 발신자 주소 |

### 테스트 발송

설정 완료 후 **"테스트 메일 발송"** 버튼으로 정상 동작을 확인합니다.

## 파일 저장소 설정

파일 업로드 저장소를 설정합니다.

| 설정 | 설명 |
|------|------|
| 저장소 유형 | 로컬 / S3 호환 |
| 최대 파일 크기 | 업로드 가능한 최대 파일 크기 |
| 허용 확장자 | 업로드 가능한 파일 확장자 목록 |

### S3 호환 저장소

| 설정 | 설명 |
|------|------|
| Endpoint | S3 엔드포인트 URL |
| Bucket | 버킷 이름 |
| Access Key | 액세스 키 |
| Secret Key | 시크릿 키 |
| Region | 리전 |

## DLP (데이터 유출 방지)

민감 정보의 유출을 방지하는 정책을 설정합니다.

### 키워드 필터

1. 관리 > 시스템 설정 > **DLP** 탭
2. 차단할 키워드 패턴을 등록합니다.
3. 메시지에 해당 패턴이 포함되면 전송이 차단됩니다.

### 파일 정책

- 특정 확장자의 파일 업로드 차단
- 외부 공유 제한

## 조직 기본 설정

| 설정 | 설명 |
|------|------|
| 조직 이름 | 화면에 표시되는 조직 이름 |
| 로고 | 사이드바 상단에 표시되는 로고 이미지 |
| 기본 언어 | 시스템 기본 언어 (한국어 / English) |
| 타임존 | 시스템 기본 시간대 |
