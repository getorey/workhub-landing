# 부서 관리

기관 관리자는 조직의 부서 구조를 설정하고 관리합니다.

## 부서 목록

관리 > **부서 관리** 메뉴에서 전체 부서 목록을 트리 형태로 확인합니다.

### 인원수 표시

각 부서 옆에 표시되는 인원수는 **하위 부서를 포함한 전체 인원수**입니다.

- 예: 경영지원본부(10) — 경영지원본부 직접 소속 + 하위 부서(인사팀, 총무팀 등) 인원 합산
- 하위 부서가 없는 경우 해당 부서의 직접 소속 인원수가 표시됩니다
- 부서를 선택하면 상세 패널에서 직접 소속 인원수와 하위 포함 전체 인원수를 함께 확인할 수 있습니다

## 부서 생성

1. **"+ 부서 추가"** 버튼을 클릭합니다.
2. 부서 이름을 입력합니다.
3. 상위 부서를 선택합니다 (최상위 부서인 경우 비워둡니다).
4. **"생성"** 버튼을 클릭합니다.

## 부서 수정

1. 부서 목록에서 대상 부서를 선택합니다.
2. 부서 이름, 상위 부서를 수정합니다.
3. 변경 사항을 저장합니다.

## 부서 삭제

1. 부서 상세에서 **"삭제"** 버튼을 클릭합니다.
2. 하위 부서가 있는 경우 삭제할 수 없습니다. 하위 부서를 먼저 이동하거나 삭제해야 합니다.
3. 소속 구성원이 있는 경우 다른 부서로 이동 후 삭제합니다.

## 부서 일괄 등록

CSV 파일을 이용하여 부서를 일괄 등록할 수 있습니다.

### 템플릿 다운로드

1. 부서 관리 화면에서 **"일괄 등록"** 버튼을 클릭합니다.
2. **"템플릿 다운로드"** 버튼으로 CSV 템플릿 파일을 다운로드합니다.

### CSV 형식

| 컬럼 | 필수 | 설명 |
|------|------|------|
| name | O | 부서 이름 |
| code | O | 부서 코드 (고유값) |
| parent_code | | 상위 부서 코드 (최상위인 경우 비워둡니다) |

**예시:**

```
name,code,parent_code
경영지원본부,MGMT,
인사팀,HR,MGMT
총무팀,GA,MGMT
개발본부,DEV,
프론트엔드팀,FE,DEV
백엔드팀,BE,DEV
```

### 미리보기

1. CSV 파일을 드래그 앤 드롭하거나 파일 선택으로 업로드합니다.
2. **미리보기** 화면에서 등록될 부서 목록을 확인합니다.
3. 오류가 있는 행은 빨간색으로 표시되며, 오류 내용이 함께 표시됩니다.

### 실행

1. 미리보기에서 오류가 없는 것을 확인한 후 **"등록 실행"** 버튼을 클릭합니다.
2. 등록 결과가 표시됩니다 (성공/실패 건수).

> **참고**: 부서 일괄 등록은 부서 구조만 변경합니다. 구성원 배정은 초대 시 부서를 지정하거나, 사용자가 프로필에서 직접 변경할 수 있습니다.

## 부서 통폐합

두 부서를 하나로 합칠 수 있습니다.

1. 부서 목록에서 통폐합할 부서를 선택합니다.
2. 상세 패널에서 **"통폐합"** 버튼을 클릭합니다.
3. 대상 부서를 선택합니다.
4. **"통합"** 버튼을 클릭하면 선택한 부서의 구성원과 하위 부서가 대상 부서로 이동되고, 원래 부서는 삭제됩니다.

::: warning
통폐합은 되돌릴 수 없습니다. 실행 전에 대상 부서를 신중하게 확인하세요.
:::
