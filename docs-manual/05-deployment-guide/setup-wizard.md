# 초기 설정 마법사

Workhub를 최초 실행하면 초기 설정 마법사가 표시됩니다.

## 설정 단계

### Step 1: 조직 정보

| 항목 | 설명 |
|------|------|
| 조직 이름 | 시스템에 표시되는 조직 이름 |
| 로고 | 사이드바에 표시되는 로고 이미지 (선택) |

### Step 2: 관리자 계정

최초 기관 관리자 계정을 생성합니다.

| 항목 | 설명 |
|------|------|
| 이름 | 관리자 이름 |
| 이메일 | 로그인 이메일 |
| 비밀번호 | 초기 비밀번호 |

### Step 3: 인증 설정

Zitadel OIDC 연동 정보를 입력합니다.

| 항목 | 설명 |
|------|------|
| OIDC Issuer URL | 인증 서버 주소 |
| Client ID | 클라이언트 ID |
| Client Secret | 클라이언트 시크릿 |

### Step 4: 완료

설정 완료 후 **"시작하기"** 버튼을 클릭하면 대시보드로 이동합니다.

## 설정 후 다음 단계

1. **부서 구조 설정**: 관리 > 부서 관리에서 조직도를 구성합니다.
2. **사용자 등록**: 관리 > 사용자 관리에서 구성원을 등록합니다.
3. **채널 생성**: 조직 공지 등 기본 채널을 생성합니다.
