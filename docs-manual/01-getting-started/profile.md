# 프로필 설정

## 프로필 확인

좌측 사이드바 하단의 사용자 아이콘을 클릭하면 프로필 페이지로 이동합니다.

## 수정 가능한 항목

| 항목 | 설명 |
|------|------|
| 표시 이름 | 메시지, 멘션 등에 표시되는 이름 |
| 프로필 사진 | 아바타 이미지 업로드 (최대 5MB) |
| 소속 부서 | 본인이 소속된 기관 내 부서 선택 |

## 프로필 사진 변경

1. 프로필 페이지에서 **"프로필 편집"** 버튼을 클릭합니다.
2. 프로필 사진 영역의 카메라 아이콘을 클릭합니다.
3. 이미지 파일을 선택합니다.
4. **"저장"** 버튼을 클릭하면 반영됩니다.

## 부서 변경

본인의 소속 부서를 직접 변경할 수 있습니다.

1. 프로필 페이지에서 **"프로필 편집"** 버튼을 클릭합니다.
2. 부서 드롭다운에서 이동할 부서를 선택합니다.
3. **"저장"** 버튼을 클릭합니다.

::: tip
부서 목록은 본인이 소속된 기관의 부서만 표시됩니다. 하위 부서가 있는 경우 계층 구조로 표시되어 쉽게 찾을 수 있습니다.
:::

> **참고**: 이메일과 역할은 기관관리자가 관리합니다.
> 변경이 필요한 경우 기관관리자에게 문의하세요.
