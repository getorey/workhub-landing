# 프로필 설정

## 프로필 확인

좌측 사이드바 하단의 사용자 아이콘을 클릭하면 프로필 페이지로 이동합니다.

## 수정 가능한 항목

| 항목 | 설명 |
|------|------|
| 표시 이름 | 메시지, 멘션 등에 표시되는 이름 |
| 프로필 사진 | 아바타 이미지 업로드 (최대 5MB) |

## 프로필 사진 변경

1. 프로필 페이지에서 사진 영역을 클릭합니다.
2. 이미지 파일을 선택합니다.
3. 업로드가 완료되면 자동 반영됩니다.

> **참고**: 이메일, 역할, 소속 부서 등은 기관관리자가 관리합니다.
> 변경이 필요한 경우 기관관리자에게 문의하세요.
