# 화면 구성

## 전체 레이아웃

Workhub는 좌측 사이드바 + 메인 콘텐츠 영역으로 구성됩니다.

### 좌측 사이드바

| 메뉴 | 설명 |
|------|------|
| 대시보드 | 전체 현황 요약 |
| 프로젝트 | 프로젝트 목록 및 관리 |
| 채널 | 팀/조직 채널 메시징 |
| DM | 1:1 및 그룹 다이렉트 메시지 |
| 연락처 | 조직 구성원 연락처 관리 |
| 조직도 | 부서 계층 구조 조회 |
| 검색 | 메시지, 채널, 프로젝트, 사용자 통합 검색 |
| 알림 | 알림 센터 |

### 관리자 전용 메뉴

기관관리자로 지정된 경우 사이드바 하단에 **관리** 메뉴가 추가됩니다:

- 사용자 관리
- 부서 관리
- 봇 관리
- 설정 (SMTP, 보안정책)
- 감사 로그

## 알림 배지

사이드바의 각 메뉴 항목에는 읽지 않은 알림 수가 배지로 표시됩니다.

## 반응형 동작

- 좌측 사이드바는 화면 너비에 따라 자동으로 축소됩니다.
- 모바일 브라우저에서도 기본 기능을 사용할 수 있습니다.
