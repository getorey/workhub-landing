# 로그인 및 첫 접속

## 로그인

1. 브라우저에서 Workhub URL에 접속합니다.
2. 조직에서 제공한 계정으로 로그인합니다.
   - **SSO 로그인**: "SSO로 로그인" 버튼을 클릭하면 조직의 인증 시스템(Zitadel)으로 이동합니다.
   - **초대 링크**: 기관관리자로부터 받은 초대 이메일의 링크를 클릭하여 계정을 생성합니다.

## 첫 접속 시

처음 로그인하면 대시보드가 표시됩니다. 대시보드에서 다음 항목을 확인할 수 있습니다:

- **프로젝트 현황**: 참여 중인 프로젝트 수
- **채널 현황**: 가입된 채널 수
- **나의 태스크**: 나에게 배정된 태스크 목록
- **최근 DM**: 최근 다이렉트 메시지
- **알림**: 읽지 않은 알림

## 비밀번호 변경

비밀번호 변경은 조직의 인증 시스템에서 관리됩니다.
SSO를 사용하는 경우 해당 인증 포털에서 비밀번호를 변경하세요.
