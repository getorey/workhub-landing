# 알림 설정

## 알림 센터

사이드바의 **알림** 메뉴에서 모든 알림을 확인할 수 있습니다.

### 알림 유형

| 유형 | 설명 |
|------|------|
| 메시지 | 참여 중인 채널/토픽의 새 메시지 |
| 멘션 | `@이름`으로 언급된 경우 |
| 태스크 배정 | 나에게 태스크가 배정된 경우 |
| 태스크 상태 | 관련 태스크의 상태가 변경된 경우 |
| DM | 새 다이렉트 메시지 |
| 프로젝트 초대 | 프로젝트에 초대된 경우 |
| 시스템 | 시스템 공지 및 안내 |

### 알림 관리

- **읽음 처리**: 알림을 클릭하면 자동으로 읽음 처리됩니다.
- **전체 읽음**: "모두 읽음" 버튼으로 일괄 처리합니다.
- **필터**: "읽지 않음만 보기"로 필터링할 수 있습니다.

## 알림 설정

사이드바 > 알림 설정 페이지에서 알림 수신 여부를 설정합니다.

| 설정 | 기본값 | 설명 |
|------|--------|------|
| 채널 메시지 | ON | 채널의 새 메시지 알림 |
| DM 메시지 | ON | DM의 새 메시지 알림 |
| @멘션 | ON | 멘션 알림 |
| 태스크 배정 | ON | 태스크 배정 알림 |
| 태스크 상태 변경 | ON | 관련 태스크 상태 변경 |
| 프로젝트 초대 | ON | 프로젝트 초대 알림 |
| 브라우저 알림 | OFF | 브라우저 푸시 알림 |
| 이메일 알림 | OFF | 이메일 알림 발송 |
