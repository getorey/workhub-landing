# 채널 관리

부서 관리자는 부서 내 채널을 생성하고 운영합니다.

## 채널 생성

1. 사이드바 > **채널** > **"+ 채널 만들기"** 버튼을 클릭합니다.
2. 채널 이름과 설명을 입력합니다.
3. 채널이 생성되면 부서 구성원이 가입할 수 있습니다.

## 채널 설정 변경

채널 상세 화면 > **설정** 탭에서 다음을 수정합니다:

- **채널 이름**: 채널의 표시 이름을 변경합니다.
- **채널 설명**: 채널의 목적과 용도를 설명합니다.

## 채널 구성원 관리

- 채널 상세 > **구성원** 탭에서 현재 참여자를 확인합니다.
- 필요시 특정 사용자를 채널에서 제거할 수 있습니다.

## 채널 보관

더 이상 사용하지 않는 채널은 보관 처리합니다.

1. 채널 설정에서 **"채널 보관"** 버튼을 클릭합니다.
2. 보관된 채널은 메시지 전송이 불가하며, 기존 메시지는 읽기 전용으로 유지됩니다.
